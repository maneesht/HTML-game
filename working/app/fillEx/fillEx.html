Remember, you can add style within body tags to add background color, font color, and more.

Also, you can add more than one type of style in the same tag by seperating them with semicolons.

<div>
<h3 style="background:black;color:white">
This is our header
</h3>
</div>

The above code will make the background black and the font color white!

Color Page

How to change background color:

Inside a body tag, type in: style="background:(type a color)". If you wish to get a specific color shade, you will need to search for the color tag which can be easily found online. This background color change will apply to everything that the body tag controls, so putting it in the outermost <div> will change the entire background. If you had a table and added the same "style" in a <td> tag, you would only change the color of that one cell.


Changing text color

This is the same concept as changing the background color. Just as before, you will be adding a style to a body tag. However, this time we will be writing a different style.


Example: style="color:red"

This will change only the font color to red, not the background.


Useful Knowledge:

When you need to add multiple styles into the same body tag, you do not write style more than once. Simply seperate each command with a semicolon.


Example: style="background:blue;text-align:center;color:white"

This will make the background color blue, align the text to the center of the page, and change the text color to white.
