The basic structure of HTML is like this:

<html>
<body>
Here goes all of your data!
</body>
</html>

All html pages start out with an html tag

You can end tags by adding a </ then the tag

HTML is NOT case-sensitive

Make sure to end your HTML tags

Other common tags that are used include <p>,<h1>,<h2>,<h3>,<h4>. The <p> tag is used to seperate text into a paragraph. For the header tags (<h1>,<h2>,<h3>,<h4>), the lower the header number value, the bigger the text will be.
